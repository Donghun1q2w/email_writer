이메일 자동 작성 도구..
환경은 윈도우 10 환경에서 구동할 예정임.
1. 본인이 작성한 이메일들을 참고하여 유사한 톤과 구조로 이메일 작성을 도와주는 도구.
2. 이메일 환경은 아웃룩 2013~2016버전.
3. 도구 작성환경은 아웃룩과 연결하는 vba, 주 로직은 파이썬으로 구축하고싶음.
4. 이메일 작성 로직은 google gemini api를 사용하여 gemini file search 시스템을 사용하고 싶음.
준비: 1. 이메일(msg)파일들을 마크다운 파일로 변환한 후 해당 파일들을 file search에 등록함. 

5. 메일 전체의 본문과 사용자가 블록으로 처리한 부분을 입력값으로 설정. 
6. 메일 전체 본문은 메일의 맥락, 사용자가 블록 처리한 부분은 수신자, 메일 작성할 요지/키워드 등임.
7. 메일이 회신일 경우 메일 본문을 참고하고, 블록을 기반으로 file search에서 기존에 작성했던 패턴을 유지하며 메일을 작성할 수 있도록 작성.
8. 제미나이에서 출력한 메일 본문은 vba를 거쳐 사용자 블록부분을 대체하고 입력.
9. 그 외 추가 프롬프트가 입력될 부분도 고려하여 로직 설계 할 것.
